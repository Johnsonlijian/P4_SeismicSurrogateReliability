# R25 reliability hardening note

Date: 2026-06-01

This public reproducibility package was updated after the R25 manuscript hardening pass. The public update contains only reproducible, non-sensitive materials:

- Python source for Structural Safety tables and false-safe reliability figure generation.
- Derived CSV tables, including event-bootstrap false-safe uncertainty columns and residual variance decomposition.
- Derived Fig.5 exports in PDF/PNG/SVG.
- Verified DOI additions for ground-motion-model event residual context.

The active submission manuscript, internal model-review reports, raw NSMP downloads and private round logs are intentionally excluded from the public repository.
