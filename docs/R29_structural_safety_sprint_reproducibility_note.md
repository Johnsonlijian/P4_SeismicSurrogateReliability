# R29 Structural Safety sprint reproducibility note

This public repository is intentionally limited to reproducibility materials. R29 updated the visible manuscript terminology from a sufficient-sounding gate to a necessary false-safe reliability filter, rebuilt the Figure 1 mechanism diagram from code, refreshed Figure 6 and Figure 7 labels, and retained R28 true event-disjoint large-budget generation scripts.

The large local prediction export contains 796,080 regenerated event-disjoint prediction rows for N=50,100,250,500,1000,2000. The repository provides scripts, aggregate summaries and derived figures; it does not publish active manuscript drafts, journal LaTeX source, internal review rounds, raw third-party archives or credentials.

Scientific boundary: recorded ground motions are public strong-motion records, while structural response labels are nonlinear numerical MDOF simulation outputs rather than field-observed structural damage.
