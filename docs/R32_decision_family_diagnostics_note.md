# R32 decision and structural-family diagnostics

This public reproducibility addendum contains the script and derived outputs for the R32 supplementary diagnostics.

- code/71_r32_decision_family_diagnostics.py

## Evidence boundary

The decision map is derived from existing R22/R24 decision-risk and false-safe reliability outputs. The structural-family diagnostic is derived from the true R28 event-disjoint N=2000 prediction export.

The structural-family diagnostic is a stratified diagnostic within the existing event-disjoint prediction export. It is not a leave-family-out retraining experiment and should not be cited as proof of broad inventory extrapolation.

Submission-facing manuscript PDFs, cover letters, reviewer materials, and active supplementary PDFs are intentionally excluded from this public repository.
